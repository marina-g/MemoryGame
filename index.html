Instructions

This is a time limited one player memory game, where a player is shown a number of images faced down. Only a set of 2 images can be opened at a time and if they are the same they will stay open, if not they will flip back. The goal is to guess the pairs in as less time and tries as possible, where one try is considered one opened set of images.

The score is determined by the number of matched pairs, the number of tries and the time left after matching all the images on the board.
